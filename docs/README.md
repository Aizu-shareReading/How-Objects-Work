## ❓問いリスト❓

- 「オブジェクト」とは？
  - 状態と操作をまとめたもの
  - Q. クラスとはどう違うの？
  - A. クラスはオブジェクトを作る手段の一つであって、オブジェクトすべてを表現していない  
  例えば、JavaScriptでは`class`を用いずともオブジェクト`{}`は作れる
